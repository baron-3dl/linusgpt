# LinusGPT

A self-roast homage to Linus Torvalds' brutal LKML code reviews — paste your *own* code, get roasted. Clearly-labeled parody; end goal is to donate the project to the Linux Foundation.

## Work tracking
This project does **not** use `rd`. Track work in **`TASKS.md`** (repo root) — the durable backlog/plan — plus Claude Code's in-session todo panel for active multi-step work. Update `TASKS.md` checkboxes as items complete.

## Key docs
- `docs/signature-interaction.md` — the locked interaction (Paste-Code Linus Roast)
- `docs/POSTURE.md` — parody/legal stance + guardrails **G1–G4**
- `docs/linusgpt-system-prompt-v3.md` — current voice (the "bazinga"); all surfaces wrap this
- `docs/voice-rubric.md`, `docs/voice-blind-test*.md` — voice spec + eval results
